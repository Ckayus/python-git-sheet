String Methods

Method Description Problem Statement Code Snippet
str.capitalize() Converts the first character to uppercase and the rest to lowercase Convert a string to title case "hello world".capitalize()
str.lower() Converts all characters to lowercase Convert a string to lowercase "HELLO".lower()
str.upper() Converts all characters to uppercase Convert a string to uppercase "hello".upper()
str.strip() Removes leading and trailing whitespace Remove whitespace from a string " hello ".strip()
str.replace(old, new) Replaces all occurrences of old substring with new Replace a substring in a string "hello world".replace("world", "python")
str.split(sep) Splits the string at the specified separator and returns a list Split a string into a list "hello,world".split(",")
str.join(iterable) Concatenates the elements of the iterable separated by the string separator Join a list of strings into one ",".join(["hello", "world"])

Dictionary Methods

Method Description Problem Statement Code Snippet
dict.get(key, default=None) Returns the value for the given key, or default if the key is not found Get the value of a key in a dictionary with a default value person = {"name": "John", "age": 30}
person.get("age")
person.get("city", "Unknown")
dict.keys() Returns a view object containing the keys of the dictionary Get a list of keys in a dictionary person = {"name": "John", "age": 30}
person.keys()
dict.values() Returns a view object containing the values of the dictionary Get a list of values in a dictionary person = {"name": "John", "age": 30}
person.values()
dict.items() Returns a view object containing the key-value pairs of the dictionary Get a list of key-value pairs in a dictionary person = {"name": "John", "age": 30}
person.items()
dict.pop(key, default=None) Removes and returns the value for the given key, or default if the key is not found Remove and return the value of a key in a dictionary person = {"name": "John", "age": 30}
person.pop("age")
dict.update(other) Updates the dictionary with the key-value pairs from other Update a dictionary with new key-value pairs person = {"name": "John", "age": 30}
person.update({"city": "New York"})

Set Methods

Method Description Problem Statement Code Snippet
set.add(x) Adds an element to the set Add an element to a set fruits = {"apple", "banana"}
fruits.add("cherry")
set.remove(x) Removes an element from the set Remove an element from a set fruits = {"apple", "banana", "cherry"}
fruits.remove("banana")
set.discard(x) Removes an element from the set if it is a member Remove an element from a set if it exists fruits = {"apple", "banana", "cherry"}
fruits.discard("banana")
set.pop() Removes and returns an arbitrary element from the set Remove and return an arbitrary element from a set fruits = {"apple", "banana", "cherry"}
fruits.pop()
set.union(other) Returns a new set containing the unique elements from both sets Find the union of two sets fruits1 = {"apple", "banana"}
fruits2 = {"banana", "cherry"}
fruits1.union(fruits2)
set.intersection(other) Returns a new set containing the common elements between the sets Find the intersection of two sets fruits1 = {"apple", "banana"}
fruits2 = {"banana", "cherry"}
fruits1.intersection(fruits2)
set.difference(other) Returns a new set containing the elements that are in the first set but not in the second Find the difference between two sets fruits1 = {"apple", "banana"}
fruits2 = {"banana", "cherry"}
fruits1.difference(fruits2)
